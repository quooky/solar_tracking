# Documentation

This folder contains documentation for the Solar Tracking project.

## Contents

- `thesis.pdf` - Bachelor's thesis document (place your thesis PDF here)
- `images/` - Screenshots and demo images for the README

## Adding Images to the Main README

After adding images to the `images/` folder, uncomment the relevant sections in the main README.md:

1. Demo GIF: `docs/images/tracking_demo.gif`
2. Results plot: `docs/images/rotation_plot.png`

Example images to include:
- A screenshot or GIF of the interactive tracking window
- The differential rotation plot output
- A sample solar image with detected sunspots highlighted
